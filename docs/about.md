Following options for search are available based on user input. If no user input is provided for any of the fields, whole range (rating) or all options (title, author, pages in book and date published) is included in resulting list of books.

Search options:<br>
- **Book title** <br>
- **Author name**<br>
- **Average rating**<br>
- **Genre**<br>
- **Page count**<br>
- **Date published**<br>

#### Book recommendations ðŸ“–<br>
Book recommendations include 3 possible searches:<br>
- **Top 10 best rated books by genre**<br>
Based on the user input genre top 10 rated books in selected genre are returned.<br>
- **Random book**<br>
Random book from the whole dataset is reruned.<br>
- **Ten random books by chosen genre**<br>
Search returns 10 random books based on genre specified by user.<br>
